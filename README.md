# Programming Tutorial — Fuzzy Cognitive Maps (FCM) | Used Car Evaluation

This repository contains my **individual submission** for the *Programming Tutorial* of the course **Fuzzy Sets and Systems**.

## What is included
- `03_UsedCar_FCM.ipynb` — A Scenario-based Fuzzy Cognitive Map notebook applied to **used-car purchase evaluation**.
- `requirements.txt` — Python dependencies needed to run the notebook.

## Notebook goal (short)
The notebook models used-car decision making under uncertainty using an **FCM** with concepts such as:
mileage, age, maintenance quality, seller trust, reliability, repair risk, value-for-money, and purchase recommendation.

It runs multiple **what-if scenarios** and compares the outcomes (plots + final summary table).
